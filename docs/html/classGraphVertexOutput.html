Detailed Description

class GraphVertexOutput It is a vertex of the graph, specially designed for data output.

It inherits from the GraphVertexBase class and adds additional functionality related to data output

◆ updateLevel()

void GraphVertexOutput::updateLevel ( )
virtual

updateLevel updates the level of the current vertex in the graph based on the levels of its incoming connections.

If a vertex has incoming connections, the method determines the highest level among all the vertices to which it is connected, and sets the level of the current vertex to one higher than the highest level

Reimplemented from GraphVertexBase.

◆ updateValue()

char GraphVertexOutput::updateValue ( )
virtual

updateValue updates the value of the current vertex of the graph based on the values of its incoming connections and the type of logical element (or "gate").

Depending on the type of gate and the values of the incoming links, the method uses the truth tables to perform the corresponding logical operation and sets a new value for the current vertex

Returns
A char that represents the new value of the current vertex after updating based on the values of its incoming connections and the type of logical element (or "gate").
