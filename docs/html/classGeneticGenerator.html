createPopulation Creates the initial genetic population The method clears the current population and then generates a new population of a given size.

For each individual in the population, a random chromosome of Type is generated using passed parameters. Individuals are added to a population and then the population is maintained

◆ generate()

template<typename Type , typename ParametersType >
std::vector<ChronosomeType<Type, ParametersType> > GeneticGenerator< Type, ParametersType >::generate ( )
inline

generate Generates a genetic population and carries out evolutionary processes The method creates an initial population, calculates the value of the process end function, and in a cycle performs the operations of recombination, mutation and selection until reaching criterion for completing a process or reaching the maximum number of cycles.

On each iteration preserves the population. Upon completion, returns the final population.

Returns
Final genetic population
