Threading::ThreadPool class
#include <ThreadPool.hpp>

Contents

  • Reference
    • Public types
    • Constructors, destructors, conversion operators

Allows to execute tasks in multithread manner. It is not allowed to use thread pool in multiple threads. It is guaranteed tasks process-start times and ordered as their add time.

void Threading::ThreadPool::stop()

Stop workers. Some tasks can be incomplete. Blocks until every thread won't be stopped and destroyed. After that, thread pool can not process new tasks.
