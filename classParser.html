bool Parser::parseAll()

parseAll Parse all logical expressions and construct the graph. This method initializes a new oriented graph and iterates through all logical expressions stored in the d_logExpressions member variable. For each expression, it checks if it has balanced brackets using the createBrackets method. If the brackets are balanced, it calls the parseToVertex method to parse the expression and construct the corresponding vertex in the graph. If parsing fails for any expression, the method returns false. Otherwise, it returns true indicating successful parsing of all expressions.
