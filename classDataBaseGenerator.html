Detailed Description

class DataBaseGenerator

Parameters
d_mainPathA string containing the path to the main database directory. The default value is ".", which means the current working directory
d_settingsA smart pointer to an object of the Settings class representing the settings for the database generator
d_parametersAn object of the DataBase Generator Parameters class containing parameters for generating a database. This object is used to configure database generation
d_dirCountAn integer value representing the number of directories created during database generation
d_randGeneratorRandom number generator with seed
