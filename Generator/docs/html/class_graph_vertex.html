Detailed Description

class GraphVertex

Parameters
d_logicExpressionStores a logical expression associated with a graph vertex
d_operationStores the operation associated with the vertex of the graph. This operation can be any operation defined for a given vertex, for example, "AND", "OR", "NOT", "XOR", etc.
d_levelThe level of the vertex in the graph
d_valueThe value of the vertex. This Boolean value represents the current state of the vertex, which can be true or false depending on the execution of the operation or the calculation of the Boolean expression.
d_wireNameThis name can be used to identify the wire or connection associated with a given vertex in the graph*
d_countThis static field is used to count the number of Graph Vertex class objects created
d_settingsA pointer to the settings object for the vertex of the graph
