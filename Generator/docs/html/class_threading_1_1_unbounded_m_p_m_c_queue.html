Detailed Description

template<typename T>
class Threading::UnboundedMPMCQueue< T >

Allows to add tasks and blocking wait for element appearance for multiple writers (producers) and multiple readers (consumers).

User must guarantee no adding in the closed queue, else runtime error exception will be thrown.
